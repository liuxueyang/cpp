示例 1:

输入: n = 4, edges = [[0,1,3],[3,1,1],[2,3,4],[0,2,2]]

输出: 5

解释:



使用路径 0 → 1 (成本 3)。
在节点 1，将原始边 3 → 1 反转为 1 → 3 并穿过它，成本为 2 * 1 = 2。
总成本为 3 + 2 = 5。
示例 2:

输入: n = 4, edges = [[0,2,1],[2,1,1],[1,3,1],[2,3,3]]

输出: 3

解释:

不需要反转。走路径 0 → 2 (成本 1)，然后 2 → 1 (成本 1)，再然后 1 → 3 (成本 1)。
总成本为 1 + 1 + 1 = 3。
